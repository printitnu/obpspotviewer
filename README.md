OBPVIEWER_MOD – UPDATED OBP VIEWER (SPOT + LINE MELTING)

This is a modified version of the obplib OBP viewer, extended for EB-PBF workflows that use both line melting and spot melting.

Main improvements:

TimedPoints (spots) are colored by dwell time

Log scaling is used for dwell to improve contrast at low ms values

Dwell is displayed in milliseconds (ms)

Optional SpotViz mode to visualize spot size

Optional multi-layer viewing from a JSON/YAML build file (not bug-free yet)
